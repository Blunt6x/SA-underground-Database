## üí° Tips & Tricks

### Quick Edit Workflow

2. Click "Edit" button

### Image Upload Tips
- Click "Browse" to select image file
- Image uploads and path is automatically filled in
- Upload confirmation appears in top-right
- You can upload multiple images in sequence

### Music Management
- Enter song title and artist name
- Click "Upload Music" to select MP3 file
- See upload progress in the song list
- Delete songs with the "Delete" button

## üêõ Troubleshooting

**Search not working?**
- Make sure you're typing in the search box
- Check that artists are loaded first
- Try clearing and searching again

**Updates not saving?**
- Check error message in top-right corner
- Verify all required fields are filled
- Check browser console for more details (F12)

**Images not uploading?**
- File must be an image (jpg, png, webp, etc.)
- Maximum file size is 5MB
- Check browser console for detailed errors
